To run this website locally yourself, you'll need to open two separate terminal windows (one for the backend and one for the frontend).

Here are the exact commands you need to enter in each:

Terminal 1: Start the Backend (FastAPI)
Open a new terminal and run these commands to activate the Python environment and start the API:

bash
cd "Desktop/Hackathons/Hackachinno 3/ParkEasy/backend"
source venv/bin/activate
uvicorn main:app --reload --port 8001
(The backend runs on http://127.0.0.1:8001)

Terminal 2: Start the Frontend (React/Vite)
Open a second terminal window and run these commands to start the React development server:

bash
cd "Desktop/Hackathons/Hackachinno 3/ParkEasy/frontend"
npm run dev
(The frontend will be available at http://localhost:5173)

Note on Node.js: Your Vite server requires Node.js version 20.19+ or 22+. Since I saw you have nvm installed, if you run into an engine error simply execute nvm install 22 && nvm use 22 in the frontend terminal before running npm run dev.

I've already installed the npm packages and the python pip libraries for you, so you don't need to run any install commands again!
